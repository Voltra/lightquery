Calling the factory is very similar to jQuery, I even kept the context part intact despite not liking it. Under the hood it uses Sizzle for CSS selectors which has a lot of useful additional pseudo selectors (the ones in jQuery).

You can also wrap the result of document.querySelectorAll for instance:
